# 🎨 CSS Fundamentals

## 📌 Overview
This module focuses on building a strong foundation in CSS by understanding how styles are applied, how layouts behave, and how elements are positioned within a webpage.

The emphasis is on **core concepts**, ensuring clarity before moving to advanced layout systems.

---

## 🎯 Topics Covered

### 1. CSS Introduction
- Syntax and structure
- Selectors (basic)
- Inline, Internal, External CSS
- Properties and values

### 2. Box Model
- Content, Padding, Border, Margin
- Box sizing and layout impact
- Spacing and element dimensions

### 3. Cascade & Specificity
- Cascade flow (order of rules)
- Specificity hierarchy
- Inline vs internal vs external priority
- Inheritance behavior

### 4. Positioning
- `static` (default behavior)
- `relative` (offset from itself)
- `absolute` (relative to parent)
- `fixed` (relative to viewport)
- `sticky` (scroll-based positioning)
- Introduction to `z-index`

---

## 🛠️ Practical Implementation
- Built structured examples for each concept  
- Tested layout behavior using different positioning strategies  
- Analyzed how CSS rules interact and override each other
